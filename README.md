# hrms
HRMS Lite is a lightweight, web-based Human Resource Management System designed to simulate a basic internal HR tool. The application enables an admin to manage employee records and track daily attendance through a clean, professional, and user-friendly interface.

This project demonstrates end-to-end full-stack development using Django, covering:

Frontend development

RESTful Backend API design

Database modeling & persistence

Server-side validation & error handling

Production-ready project structure

Employee Management

The admin can:

Add a new employee with:

Employee ID (unique)

Full Name

Email Address

Department

View a list of all employees

Delete an employee

Attendance Management

The admin can:

Mark attendance for an employee with:

Date

Status (Present / Absent)

View attendance records per employee

Tech Stack
Backend:

Django (Python)

Django REST Framework (DRF)

Database (Choose one):

SQLite (default for development)
or

PostgreSQL / MySQL (for production)

Frontend:

HTML, CSS, JavaScript

Django Templates

Bootstrap (for clean UI)
